# Phone Store

This is a phone store website built with Vite, TypeScript, and Tailwind CSS.

## Technologies Used

*   Vite
*   TypeScript
*   Tailwind CSS

## Project Structure

```
project/
├── index.html
├── src/

│   ├── types.ts
│   ├── utils.ts

│   └── components/
│       ├── header.ts
│       ├── footer.ts
│       ├── product-card.ts
│       ├── product-list.ts
│       ├── search-bar.ts
│       └── login-form.ts
├── public/
├── package.json

```

## Getting Started

1.  Clone the repository: `git clone <repository-url>`
2.  Install dependencies: `npm install`
3.  Run the development server: `npm run dev`
4.  Build for production: `npm run build`

## Key Features

*   **Product Catalog:** Browse a wide selection of phones.
*   **Search and Filtering:** Easily find the phone you're looking for.
*   **User Login:** Securely log in with your email and password.
*   **Dummy Payment:** Simulate the checkout process with dummy payment information.
